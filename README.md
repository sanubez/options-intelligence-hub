# Options Intelligence Hub v4.0

Advanced options analysis dashboard for Delta Exchange with:
- IV Crush/Spike Detection
- Straddle Analysis
- Market Regime Classification
- Volatility Analysis (IV-RV Framework)
- Trading Journal

## Features
- Real-time data from Delta Exchange API
- Percentile-based OI analysis
- EMA-based market structure
- Auto-refresh capability
- Mobile-responsive design

## Usage
Select asset (BTC/ETH/SOL) and expiry date from sidebar to analyze options chain.
